Bracket Balance Checker (Java)
------------------------------------------------------------------------------------------------------------------

Description

This Java program checks whether curly brackets {} in a string are balanced and correctly ordered.

It validates logical correctness, not just count — ensuring every opening bracket has a matching closing one.
--------------------------------------------------------------------------------------------------------------------

 Features

Checks correct opening–closing order

Detects early closing errors

Handles missing brackets

User-friendly console messages

Unit tests included
-------------------------------------------------------------------------------------------------------------------

 Core Logic

Iterates through characters

Counts { and }

Ensures closing brackets never exceed opening ones

Final count validation
